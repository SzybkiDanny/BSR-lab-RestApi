Required

  • Node.js
  • npm (bundled with Node.js)
  • MongoDB

Optional

  • Git
  • Postman
  • Postman
  • nodemon (requires npm): npm install -g nodemon

GitHub repository

git clone https://github.com/SzybkiDanny/BSR-lab-RestApi

CLI

Installing nodemon

monitor script for development
npm install -g nodemon

Installing node packages

execute in the project directory: ../BSR-lab-RestApi >
npm install

Running MongoDB

execute in the project directory: ../mondgodb >
(create ../data/db directory first)
md data/db
bin\mongod --dbpath ./data/db/

Running server with node

execute in the project directory: ../BSR-lab-RestApi >
node server.js

Running server with nodemon

execute in the project directory: ../BSR-lab-RestApi >
nodemon server.js

Project structure

            - public                    holds files for frontend application
            ----- app.js                application main controller
            ----- index.html            application main view
            - server                    holds files for backend application
            ----- models                holds db models
            --------- user.js           user model
            --------- announcement.js   announcement model
            - package.json              packages to install
            - server.js                 server implementation
